# Mage.jl
Mage.jl is a WIP Magic card game simulation engine written in pure Julia. 

**Features**
- EDH (Commander) support
- Deck Builder
- Seam carving for custom card images
- GIFs as custom cards
- Basic resource counter
- Game zone card counter
- Custom animations


__Known issues__
- No audio on linux
- EDH support only


__Future goals__
- Multiplayer support
- Rules enforcement
